centos安装lamp环境

  • 记录

今天在一台新机器上安装lamp环境，
本来想装一个xampp，省事。
但是想到这机器可能以后还是要跑生产环境，
还是老老实实手动配置

机器上跑的是centos6.7，
首先把源换成网易源（嘿嘿嘿，大家都懂得）

  • 首先备份/etc/yum.repos.d/CentOS-Base.repo

yum clean all
yum makecache

tar zcvf mirror
cd php-7.0-3

./configure
make
make install

./configure –with-apxs2
make
make install


DirectoryIndex index.html


DirectoryIndex index.html index.htm index.php
```

整个环境基本配置好了
